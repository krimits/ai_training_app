    📥 INSTALLATION
    
    1. Clone repository:
       $ git clone https://github.com/krimits/ai_training_app.git
       
    2. Navigate to directory:
       $ cd ai_training_app
       
    3. Install dependencies:
       $ pip install -r requirements.txt
       
    4. Run application:
       $ streamlit run ai_training_app.py
